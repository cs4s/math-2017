# Networks Concepts

## About

In this activity, you will learn about the following *Networks* concepts:

- *Graphs, Vertices and Edges*
- *Adjacency matrices*
- *Complete graphs*
- *Subgraphs*
- *Connectedness*
- *Cycles*
- *Depth-first search*
